# googleBooksSearch
A React-based Google Books Search app
 
## Built With

* MongoDB
* Node.js
* ReactJS
* JavaScript
* HTML
* CSS
* Mongoose
* Express.js
* morgan
* Bootstrap

## Requirements

* Requires at minimum 2 pages
* Connect to a MongoDB database
* Create a Book schema
* Hide and show your React components without changing the route within Express
